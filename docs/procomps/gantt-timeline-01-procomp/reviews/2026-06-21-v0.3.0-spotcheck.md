### Smoke checklist — reasoned (live run = F-01)
1. **Bracket-drag shifts subtree** — `tryStartEdit` hits `[data-summaryid]` → `groupmove` drag → `moveEdit` computes snapped `deltaMs` → `commitEdit` → `ctx.moveSubtree` → `shiftSubtree` rewrites every leaf's `startAt`(+`expireAt`) → `onChange`. Bracket re-derives from moved leaves on re-render. ✔ paths wired.
2. **Snap + Alt-free** — `snapUnitRef` set at drag-start from `e.altKey` (parity with single-move). ✔
3. **Atomic deny → pan** — `canGroupMove` false (locked/denied leaf) ⇒ `tryStartEdit` returns false ⇒ v1 pan; `moveSubtree`'s leaf loop fires `onPermissionDenied("drag", leafId, reason)` for the imperative path. ✔
4. **`editable={false}`** — `tryStartEdit` early-returns false; `groupMovable` is `editable && …` ⇒ false ⇒ no grab cursor/hit-pad; byte-identical v1. ✔
5. **Empty summary-row area (not bracket)** — `[data-summaryid]` miss ⇒ `[data-rowid]` ⇒ `isSummary` ⇒ pan (v0.2.1). ✔
6. **Collapsed summary** — `getItem`/`subtreeLeaves` walk the full forest subtree regardless of collapse ⇒ hidden descendants shift. ✔
